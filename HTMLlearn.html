HTML

Introduction to HTML

An HTML element is defined by a start tag, some content (optional), and an end tag. The HTML element is everything from the start tag to the end tag.
.
Each of these elements start with an opening angle bracket (the less than symbol) and end with a backslash and closing angle bracket (the greater than symbol) Within these angle brackets, you can add your content.

A few common elements include...

h1, h2, h3, h4, h5, h6: "h" indicates a heading, and the number after indicates the size (ranging from 1-6, 1 being the largest)


"p" indicates a paragraph, which is generally styled to be smaller than the heading tag (though you can customise it)

"img" indicates an image- in this tag, rather than writing after closing the tag, it is written within the tag.
Here, you can add details such as the source (src), size (width, height), alternate text (alt- for when image can't be displayed), etc.
