CSS

Introduction to CSS

CSS is the language we use to style a Web page, including the design, layout and variations in display for different devices and screen sizes.
.
This language allows you to customise each element, class/id (group), etc. separately based on your wants and likes

CSS Syntax

Selector + {property:value; property: value; ....}
.
For elements...

In order to customise a particular element, you simply write down the element (such as h1),
open the curly brackets, select the property- the options come as a dropdown in most editors (eg color, font-size, font-family, margin, padding, etc.)
After each property, add a colon (:), the value (eg a specific color if the property is font-color), and end it with a semi-colon (;) to begin a new property. To end the element's styling, add a closing curly bracket, and begin writing for the next element from the next line onwards.

For classes...

In order to customise a particular class, you simply write down the class along with period (.) - eg .nav,
and style your code as you would for elements

For IDs...

In order to customise a particular ID, you simply write down the class along with hashtag (#) - eg #button,
and style your code as you would for elements

To specifiy further...

After the element/class/ID written, add a space and add the element/class/ID within the previous category.
This allows you to customise the specific codes within a certain category in a category.
For instance, you can customise the heading tag in the navigation bar class (.nav h1) differently from the one in the text box class (.textbox h1).

A few common properties include...

  • Color (font-color, background-color)
    "h" indicates a heading, and the number after indicates the size (ranging from 1-6, 1 being the largest)

  • Font (font-color, font-family, font-size)
    "p" indicates a paragraph, which is generally styled to be smaller than the heading tag (though you can customise it)

  • Spacing and Sizing (padding, margin, margin-left, text-align, position, length, width)
    "img" indicates an image- in this tag, rather than writing after closing the tag, it is written within the tag.
    Here, you can add details such as the source (src), size (width, height), alternate text (alt- for when image can't be displayed), etc.
